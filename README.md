# 🤖 AI Desktop Assistant

An intelligent AI-powered desktop assistant that can control your system, answer questions from documents, and interact using voice or text.

---

## 🚀 Features

- 🎤 Voice + Text Input
- 💻 Dynamic OS Command Execution
- 🤖 AI Reasoning (Mistral LLM)
- 📚 RAG (Document-based Q&A)
- 🧠 Conversation Memory
- 🖥️ Professional Streamlit UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Mistral AI
- LangChain
- ChromaDB
- SpeechRecognition

---

## ⚡ How to Run

```bash
git clone https://github.com/your-username/ai-assistant.git
cd ai-assistant

pip install -r requirements.txt
streamlit run ui.py
```

---

## 🔐 Environment Setup

Create a `.env` file:

```
MISTRAL_API_KEY=your_api_key_here
```
